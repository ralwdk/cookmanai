#  CookmanAI

CookmanAI is a cross-platform mobile app built with **.NET MAUI** that helps you clear out your fridge. Just plug in the ingredients and cooking tools you have on hand, and the app fetches tailored, categorized recipes from across the web.

##  The MVP Goal

The core mission for version 1.0 is solving the "what's for dinner?" problem with zero waste:

* **Smart Input:** Quick entry for available ingredients and kitchen tools.
* **Recipe API Integration:** Fetches recipes from online recipe sources.
* **Categorized Results:** Groups suggestions (e.g., *Quick & Easy, Baking, One-Pot Meals*).
* **Rich UI:** Displays recipe cards with images, estimated times, and direct source links.

---

##  Tech Stack & Workflow

* **Framework:** .NET MAUI (.NET 8/9)
* **Task Management:** GitHub Projects (Kanban style)
* **Branching Strategy:** Feature branching via Pull Requests (PRs)
* **CI/CD:** GitHub Actions automatically builds and verifies pull requests before merge.
